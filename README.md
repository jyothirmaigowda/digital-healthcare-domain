# digital-healthcare-domain
Secure platform for storing, managing, and sharing patient medical records with role-based access.
# 🏥 Digital Health Record Portal

## 📌 Problem Statement
Medical records are scattered across hospitals, making it difficult for patients to access their complete health history.

## 💡 Solution
A secure centralized platform where patients and doctors can store, access, and manage medical records efficiently.

## 🎯 Features
- Secure login (Patient & Doctor)
- Upload & view medical records
- Categorization (reports, prescriptions)
- Search & filter history
- Role-based access control

## 🚀 Unique Features
- Patient-controlled data access
- QR-based record sharing
- Emergency access mode
- Smart health summary

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js / Firebase
- Database: MongoDB / Firebase
